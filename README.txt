## Traffic Fatality Project

In data directory, it contains both raw data and processed data. If you choose to download data using “0-download.R” code, please remember to save “population_md_post2010.csv” as “./data/raw_data/population/population_md_post2010.csv”. 


The Supplementary document.Rmd is the R markdown file for supplementary document and Supplementary document.pdf is the pdf version.


final code.Rmd is the R markdown file for all code, including downloading data. Again, please remember to add “./data/raw_data/population/population_md_post2010.csv” before doing further analysis.
